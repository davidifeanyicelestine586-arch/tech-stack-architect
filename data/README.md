# Ediccrew Tech Stack Architect - Data Layer

This directory contains the database and configuration files used by the Tech Stack Architect.

## Files

- **[components.json](file:///data/data/com.termux/files/home/storage/shared/tech-stack-architect/data/components.json)**: The component database detailing capabilities, categories, dependencies, conflict rules, pin requirements, difficulty, complexity, and other metadata.
- **[domain.json](file:///data/data/com.termux/files/home/storage/shared/tech-stack-architect/data/domain.json)**: The domain registry defining Web & SaaS, AI Automation, and Hardware & Mechatronics domains.
- **[recipes.json](file:///data/data/com.termux/files/home/storage/shared/tech-stack-architect/data/recipes.json)**: Pre-configured technology blueprints matching specific engineering goals.
